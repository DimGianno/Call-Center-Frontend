# Call Center Dashboard - Project Description

## Overview

Call Center Dashboard is a frontend web application for managing call center activity in a clear and organized way. It helps users review active and archived calls, inspect call details, add notes, and manage daily call workflows from one focused dashboard.

The project is built as a production-style React application connected to a deployed backend API.

## Main Features

- User authentication with login and signup flows
- Protected dashboard access for authenticated users
- Active and archived call views
- Detailed call modal with full call information and notes
- Add notes to individual calls
- Archive, unarchive, and delete call records
- Bulk archive and unarchive actions
- Search calls by phone number
- Filter calls by type, direction, date range, and duration
- Paginated call feed for larger datasets
- Statistics cards for call totals and call types
- Light and dark theme support
- Confirmation dialogs, loading states, empty states, and toast notifications

## Tech Stack

- React
- TypeScript
- Vite
- React Router
- Vitest
- React Testing Library
- Deployed frontend on Vercel
- Deployed backend API on Render

## What This Project Demonstrates

This project demonstrates the ability to build a complete frontend application with real API integration, authenticated routing, state management through custom hooks, reusable components, user feedback flows, error handling, and automated tests.

The focus was on creating a practical dashboard that feels close to a real business tool: structured, maintainable, responsive, and easy to use for everyday call management tasks.
